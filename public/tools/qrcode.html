工具介绍

二维码（Quick Response Code，快速响应矩阵码）最初由日本的Denso Wave公司于1994年发明，用于汽车制造厂的零部件追踪。如今，二维码已经渗透到了生活的方方面面：扫码支付、扫码加好友、扫码连WiFi、扫码看菜单、扫码下载App……几乎每个智能手机用户每天都在使用二维码。

这款在线二维码生成器可以帮助你免费生成各种类型的二维码。只需输入文本内容、网址链接、联系方式等信息，即可即时生成高清二维码图片。生成的二维码支持自定义尺寸（像素大小）和前景色/背景色，你可以根据品牌需求或使用场景定制二维码的外观风格。

生成后的二维码可以直接下载为PNG图片保存到本地，也可以直接打印使用。工具使用成熟的QR Code编码算法，确保生成的二维码可以被各类扫码应用（微信、支付宝、手机相机、专业扫描器等）正确识别。所有生成过程在浏览器端完成，你的输入内容不会被上传到服务器。

相关知识

二维码是一种二维矩阵条形码，由黑色模块排列在白色背景上的正方形图案组成。QR Code有4个纠错等级：L级可恢复7%的数据损失，M级15%，Q级25%，H级30%。这意味着即使二维码部分被遮挡或污损，仍可能被正确读取。二维码角落的三个"回"字形图案是定位标志，帮助扫描器确定方向和位置。最大的QR Code Version 40可以容纳多达2953字节的数字数据或4296字节的字母数字数据。除了QR Code，常见的二维码格式还有Data Matrix、Aztec Code和PDF417等。
