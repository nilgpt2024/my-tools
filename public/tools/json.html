相关知识

JSON全称为JavaScript Object Notation，由Douglas Crockford于2001年提出并推广。它基于JavaScript编程语言的一个子集，但语言无关，几乎所有现代编程语言都提供了JSON解析库。JSON的MIME类型是application/json，文件扩展名为.json。与XML相比，JSON更轻量、解析更快，因此在RESTful API和Web开发中已逐渐取代XML成为主流数据交换格式。
